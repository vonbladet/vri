V.R.I.

  • The Virtual Radio Interferometer (VRI) applet has been written by Nuria McKay, Derek McKay and Mark Wieringa.
  • Introduction - a general description of VRI.
  • Guide to VRI - a document for newcomers to radio interferometry.
  • For technical information on how to use it, please read the documentation.
  • The Java class files may be downloaded (as a compressed tar file) for use as a standalone program.
